# How to use this program

### Build
    -$: mvn clean package

### Run with java
    /**
        java -jar target [file] [page]
    */

    // e.g:
    //
    -$: java -jar target/qrpdfscanner-1.0-SNAPSHOT-jar-with-dependencies.jar where-is-pad-folder/file-pdf.pdf 1
